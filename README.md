1. Назначение и цели проекта

Название сайта: Harmony
Домен: harmony-massage.ru
Текущий URL (разработка/тест): https://eosin-tau-76.vercel.app

Что это за сайт: Это официальный сайт студии классического и оздоровительного массажа «Harmony». Сайт выполняет функции визитной карточки, информационного портала и инструмента для привлечения новых клиентов. Основная цель — конверсия посетителя в клиента через онлайн-запись на прием.

Для кого он:

· Основная аудитория: Жители Москвы, преимущественно от 25 лет и старше, испытывающие стресс, напряжение, усталость, нуждающиеся в качественном расслаблении, восстановлении после физических нагрузок или комплексном SPA-уходе.
· Вторичная аудитория: Люди, ищущие подарок (например, сертификат на массаж), или корпоративные клиенты.
· Ценностное предложение: Профессионализм (массажисты с мед. образованием), использование натуральных продуктов, уютная атмосфера и индивидуальный подход.

2. Функциональные требования и описание работы сайта

Сайт является одностраничным (лендингом) и работает по следующему принципу:

Структура и пользовательский сценарий:

1. Запись онлайн (Главный призыв к действию): Кнопка «Записаться онлайн» в самом верху страницы — это ключевой элемент. При нажатии она должна вести на форму бронирования или на отдельную страницу с расписанием и выбором услуги.
2. Информирование: Посетитель последовательно знакомится с преимуществами студии, перечнем услуг, причинами для выбора и контактной информацией.
3. Принятие решения: Подробные описания услуг с указанием цены помогают пользователю выбрать подходящий вариант. Блок «Почему выбирают нас» снимает возможные сомнения (качество специалистов, экологичность, гибкие условия оплаты).
4. Конверсия или контакт: После ознакомления с информацией пользователь либо возвращается к кнопке записи, либо использует контактные данные (адрес, телефон, email) для прямого обращения.

Контент-блоки и их функции (по текущей структуре):

· Заголовок (Hero-секция): Призыв к действию и краткое описание ценностей.
· Наши услуги: Каталог с ценами. Каждая услуга должна иметь отдельную страницу с детальным описанием (продолжительность, методика, эффект).
· Почему выбирают нас: УТП (уникальное торговое предложение) для убеждения.
· Контакты: Необходимая информация для тех, кто предпочитает звонить или приезжать без записи.

3. Особенности и уникальные характеристики

1. Четкая ориентация на целевую аудиторию: Контент говорит на языке клиентов, делая акцент на решении их проблем: «снятие напряжения», «релаксация», «восстановление для тела и души».
2. Профессионализм и доверие: Упор на медицинское образование массажистов и использование натуральных масел — это ключевые доводы для принятия решения.
3. Минималистичный и уютный дизайн (предполагается): Название «Harmony» и описание «уютная атмосфера» предполагают, что дизайн должен быть спокойным, с гармоничной цветовой палитрой (приглушенные, натуральные тона), качественными фотографиями интерьера и процессов, визуально передающими расслабление.

4. Недостатки текущей версии и рекомендации по доработке

Текущая версия — это черновой каркас (прототип) без дизайна и ключевых функций.

· Нет визуального контента (картинок): Критически важно добавить:
  · Фотографии интерьера студии (чистота, уют, атмосфера).
  · Фотографии процессов (массаж с использованием масел, релаксация).
  · Фотографии результатов (довольные, расслабленные клиенты, счастливые лица).
  · Фотографии команды (профессиональные и дружелюбные массажисты).
  · Логотип и фирменный стиль.
· Недоработана главная функция — запись онлайн: Кнопка «Записаться онлайн» сейчас никуда не ведет. Необходимо:
  · Внедрить онлайн-систему бронирования (виджеты типа Яндекс.Бронирования, Simplybook или аналоги).
  · Реализовать возможность онлайн-оплаты для повышения конверсии.
  · Добавить форму обратного звонка.
· Технические и информационные улучшения:
  · Добавить страницу «О нас» с историей и философией студии.
  · Создать раздел «Специалисты» с портфолио массажистов.
· Реализовать раздел «Акции и сертификаты».
  · Добавить карту (Яндекс.Карты/Google Maps) в блок контактов.
  · Сделать сайт адаптивным для мобильных устройств.
  · Настроить SEO (мета-теги, описание услуг, региональная привязка к Москве).

Сайт на Vercel не предоставляет исходники автоматически. Чтобы получить код, у вас должны быть:

Шаг 2: Если у вас есть исходный код

Вариант A: Сайт на React/Next.js  
1. Установите Node.js (v18+ рекомендуется).  
2. Распакуйте архив (если скачали ZIP).  
3. Откройте терминал в папке проекта:  
   ```bash
   npm install
   npm run dev
   ```  
4. Сайт запустится локально (обычно на http://localhost:3000).

Вариант B: Статический HTML/CSS/JS  
1. Распакуйте архив.  
2. Откройте файл index.html в браузере.  
   - Или используйте локальный сервер, чтобы избежать CORS:  
     ```bash
     npx serve
     ```  
     (или python -m http.server 8000)

Шаг 3: Внесение изменений и развёртывание  
1. Измените код (HTML, CSS, JS или компоненты React).  
2. Соберите проект (если используется фреймворк):  
   ```bash
   npm run build
   ```  
3. Загрузите изменения в Git.  
4. Vercel автоматически задеплоит новую версию (если подключён CI/CD).
